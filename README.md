# 📈 Predictive Analytics Using Historical Data

## 📌 Overview

This project demonstrates predictive analytics using historical sales data in Microsoft Excel. The objective is to identify sales trends and forecast future performance using historical records.

---

## 🎯 Objectives

- Analyze historical sales data.
- Identify sales trends over time.
- Forecast future sales performance.
- Generate business insights for decision-making.

---

## 🛠️ Tools Used

- Microsoft Excel
- Forecast Functions
- Pivot Tables
- Charts
- Trend Analysis

---

## 📂 Dataset

The dataset includes:

- Order Date
- Product
- Region
- Sales
- Profit
- Quantity

---

## 📊 Dashboard Features

- Total Historical Sales
- Monthly Sales Trend
- Forecasted Sales
- Regional Sales Analysis
- Product Performance
- Interactive Filters

---

## 📈 Key Insights

- Identified seasonal sales trends.
- Forecasted future revenue using historical data.
- Compared regional performance.
- Highlighted top-performing products.
